## 하루기록소 (Haru Record) (Team Project)

- [Summary] An AI-based diary application that uploads one photo a day, generates questions about that photo, and allows users to answer them. Still working on it. Available on AppStore!
- [Main Features] Frontend/backend integration structure using Firebase, push notification using fcmToken, deep link function
- [Tech Stack] Next.js, Typescript, TailwindCSS, Firebase, OpenAi api
- [Roles] Web UI/UX work, App push notification/deep link work, DB CRUD work
